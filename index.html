Data Cleaning & Exploration in SQL

Global Life Expectancy

In this project, I used SQL Server to clean and explore a global dataset on life expectancy. I removed duplicates, standardized data entries, and handled missing values by imputing life expectancy and status using logical calculations.

Through exploratory analysis, I identified trends such as the positive correlation between GDP and life expectancy, significant gaps between developed and developing countries, and cumulative changes in adult mortality over time using window functions.

This project highlighted key global health insights while showcasing my ability to transform raw data into actionable findings.

Data Exploration in SQL

In this project, I conducted exploratory data analysis on US household income using SQL to uncover trends and patterns. Key insights included identifying states and cities with the highest and lowest average household incomes, analyzing variations in income based on land and water coverage, and exploring income differences across community types. By aggregating and ranking income metrics such as mean and median, the analysis highlighted regional disparities and provided a deeper understanding of income distribution across the United States.
